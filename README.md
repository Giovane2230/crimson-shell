# Crimson Shell
A beautiful, usable, and customizable shell built with QuickShell.
## Project Status
Early development - This project is literally just starting. Expect lots of changes and additions!
### Vision
Crimson Shell aims to be a modern desktop shell that combines:

### Beautiful and clean interface
- Easy to use daily
- Highly customizable to fit your workflow

#### Built With
 QuickShell - A framework for building custom desktop shells.
### Roadmap

 - [ ] Basic QuickShell configuration
 - [ ] Top/bottom bar implementation
 - [ ] Essential widgets (clock, system tray, workspace switcher)
 - [ ] Theme system
 - [ ] Configuration files
 - [ ] Customization documentation
 - [ ] Plugin system
 - [ ] Window management integration
 - [ ] Notification system
 - [ ] System monitor widgets
